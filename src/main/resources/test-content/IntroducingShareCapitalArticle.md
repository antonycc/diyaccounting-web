# trailingBody
<ul>
    <li>Cash Account - Receipts Section of the first month - Input the value of shares (&pound;1.00) in Cell F6 using code DL in E6.</li>
    <li>
        <div>Company Secretary - Register of Members - Input the value of Share in G3, e.g. &pound;1</div>
    </li>
    <li>
        <div>Then do a manual entry on your Trial Balance, by first unprotecting the sheet, then inputting the following:&nbsp;</div>
    </li>
    <li>
        <div>Financial Accounts - Trial Balance, Month used above - Column M (Monthly Accrual) - M39 - Value of Share (positive), e.g. &pound;1</div>
    </li>
    <li>
        <div>Financial Accounts - Trial Balance, Month used above - Column M (Monthly Accrual) - M42 - Minus Value of Share (negative), e.g. -&pound;1&nbsp;</div>
    </li>
</ul>
